# Deep Strata 深空底层 — 实训设计文档

## 一、项目概述

**项目名称:** Deep Strata 深空底层
**开发环境:** Qt 5.12+ / C++11 / qmake
**游戏类型:** 地底生存建造（参考《缺氧》）
**项目周期:** Qt C++ 期末实训

### 游戏背景

深空采矿飞船失事，幸存者被困辐射异星。地表无法停留，只能纵向挖掘多层地下岩层搭建生存基地，依靠地底资源维持生存。

### 核心技术特点

1. 基于 QGraphicsView/QGraphicsScene 框架的游戏场景管理
2. 程序化纹理生成（QPainter 绘制 32×32 像素图，零外部素材依赖）
3. 单例模式管理四大核心模块（MapManager / BuildManager / SurviveData / EventManager）
4. BFS 算法实现电力连通判定
5. JSON 格式实现游戏存档/读档
6. 可视区域懒加载裁剪，优化大地图渲染性能

## 二、模块设计

| 模块 | 文件 | 功能 |
|------|------|------|
| 地图模块 | mapmanager.h/cpp | 60×80三层地层生成、Tile结构体、地块查询/挖掘、坐标转换 |
| 建造模块 | buildmanager.h/cpp | 6种建筑放置/移除、BFS电力连通判定、资源库存管理 |
| 生存数据 | survivedata.h/cpp | 氧气/体温/疲劳/辐射四项数值、工作状态切换、死亡判定 |

## 三、数据结构

### Tile 地块
```
struct Tile {
    enum Type { Empty, Dirt, Rock, Crystal, Bedrock,
                IronOre, WaterCrystal, DarkCrystal };
    int type, durability;
    float temperature;
    int resourceAmount;
};
```

- 地图：60宽 × 80深，每格32×32像素
- 三层地层：浅层岩土(0-25) / 结晶层(26-55) / 黑暗核心(56-79)

### Building 建筑
```
struct Building {
    enum Type { Storage, Generator, OxygenMaker,
                Decompressor, GrowPod, PowerPole };
    int gridX, gridY, width, height;
    bool powered, active;
};
```

| 建筑 | 电力 | 尺寸 | 功能 |
|------|------|------|------|
| 储物箱 | 0 | 1×1 | 资源存储 |
| 发电机 | +50 | 2×1 | 产出电力 |
| 制氧机 | -20 | 2×2 | 半径6格+5 O₂/秒 |
| 减压器 | -10 | 1×1 | 半径5格-1.5 辐射/秒 |
| 种植舱 | -5 | 2×2 | 降低疲劳累积 |
| 电线杆 | 0 | 1×1 | 电力中继(≤3格连通) |

### 生存数值
- **氧气 O₂** (0-100): 自然-0.5/秒，制氧机恢复，≤0持续3秒死亡
- **体温** (0-100): 向地块温度靠拢，≤10或≥90死亡
- **疲劳** (0-100): 持续累积，≥100移速减半
- **辐射** (0-100): 深层递增，≥80开始扣氧

## 五、游戏循环 (60 FPS)

```
每帧执行:
  1. EventManager 事件逻辑（每秒一次：温度/压力/怪物/水晶）
  2. SurviveData::update(dt) 生存数值更新
  3. 键盘移动处理 + 碰撞检测
  4. 怪物接触伤害判定
  5. View 平滑跟随玩家（lerp 插值）
  6. Tile 懒加载裁剪（视野外回收，视野内创建）
  7. 怪物/建筑 Item 同步
  8. HUD 面板刷新
```

## 六、验收要点

1. Qt 5.12+ 编译通过，零第三方库依赖
2. 三层地层按概率分布生成正确
3. WASD移动 + View平滑跟随
4. 挖掘/建造/电力连通功能正常
5. 四项生存数值实时变化，致死触发Game Over弹窗
6. JSON存档/读档：地块、建筑、资源、生存数值完整恢复
7. 怪物生成、曼哈顿贪心移动、接触伤害、击杀掉落
8. 全屏切换正常
9. 60FPS稳定运行无明显卡顿
