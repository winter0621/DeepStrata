项目简介 
Deep Strata 深空底层是基于 Qt5.12 + C++11 + qmake 开发的地底生存建造类实训游戏，灵感参考《缺氧》。 故事背景 深空采矿飞船失事，幸存者被困辐射异星，无法在地表生存，只能向下挖掘多层地层搭建地下基地，依靠矿产、电力、供氧设施维持生存。 本项目为 C++ Qt 期末实训作品，全程使用 QPainter 程序化生成 32×32 像素贴图，零外部图片素材依赖，内置地层生成、电力网络、生存数值、随机事件、怪物 AI、JSON 存档读档全套游戏逻辑。

地层地图系统 地图尺寸：60 格宽 × 80 格深，单格 32×32 像素 三层分层生成：浅层岩土 (0-25) / 结晶层 (26-55) / 黑暗核心 (56-79) 地块类型：泥土、岩石、基岩、铁矿、水晶、水晶体、暗水晶 地块耐久、温度、资源储量独立计算，支持挖掘消耗耐久获取资源
建造与电力系统 共 6 种可建造建筑，依靠 BFS 广度优先搜索判定电力连通范围：
建筑 电力消耗 / 产出 功能 储物箱 0 存储挖掘产出资源 发电机 +50 电力 持续产出电力 制氧机 -20 电力 范围供氧，恢复玩家氧气值 减压器 -10 电力 降低区域辐射数值 种植舱 -5 电力 减缓玩家疲劳增长速度 电线杆 0 电力中继，延长供电链路 3. 四大生存数值（实时动态变化） 氧气 O₂：每秒自然衰减，归零持续 3 秒触发死亡 体温：跟随地块温度浮动，低于 10 或高于 90 直接死亡 疲劳：持续累积，满值后移动速度减半 辐射：越深地层辐射越高，辐射≥80 时持续消耗氧气 4. 随机事件系统（每秒自动刷新） 地层温度随机波动 岩层破损（耐久下降） 地下怪物生成、自动寻路攻击玩家 地底水晶自然生长，产出额外资源 5. 游戏操作快捷键
